The W3C AI Knowledge Representation Community Group (AI KR CG) develops vocabularies, ontologies, and technical notes for representing knowledge in AI systems, with a focus on agent interoperability, semantic alignment, and standards readiness across the Web ecosystem.

This page lists published and in-progress deliverables. All documents are Community Group Drafts unless otherwise noted. They are not W3C Standards and are not on the W3C Standards Track.

Deliverables

  • Agent Ontology Alignment: IDL Agent Ontology and HMAS Ontology Stack CG-Draft
    Editor: Paola Di Maio  •  Published: 8 March 2026
    Formal method for ontology alignment with a use-case mapping between the IDL Agent Ontology and the HMAS Core, Interaction, and Regulation ontologies. Defines semantic correspondences (subClassOf, equivalentClass, property mappings) for agent system interoperability.
  • Voice AI Framework: Requirements and Design Principles Editor's Draft
    Editor: Paola Di Maio  •  In progress
    Analysis of documented failures in voice AI deployments and a proposed configurable framework for voice AI user experience. Informed by the W3C Smart Agents Workshop (February 2026).
  • WebMCP Technical Notes Editor's Draft
    Editor: Paola Di Maio  •  In progress
    Technical notes on the W3C Web Machine Learning (WebMCP) specification, covering security, model cards, interoperability, and quality infrastructure for MCP-compliant systems.
