W3C AI Knowledge Representation
Community Group

w3.org/community/aikr  •  GitHub repository  •  Mailing list

The W3C AI Knowledge Representation Community Group (AI KR CG) develops vocabularies, ontologies, and technical notes for representing knowledge in AI systems, with a focus on agent interoperability, semantic alignment, and standards readiness across the Web ecosystem.

Documents listed below are Community Group Drafts unless otherwise noted. They are not W3C Standards and are not on the W3C Standards Track.

Deliverables

Agent Ontology Alignment

  • Agent Ontology Alignment: IDL Agent Ontology and HMAS Ontology Stack CG-Draft
    Editor: Paola Di Maio  •  8 March 2026
    Formal method for ontology alignment with a use-case mapping between the IDL Agent Ontology and the HMAS Core, Interaction, and Regulation ontologies. Defines semantic correspondences (subClassOf, equivalentClass, property mappings) for agent system interoperability.
    Source: github.com/w3c-cg/aikr/blob/main/agonalign/index.html

Voice AI

  • Voice AI Standards -- ReSpec Draft Editor's Draft
    Editor: Paola Di Maio  •  In progress
    Requirements and design principles for voice AI systems, with analysis of documented deployment failures and a proposed configurable framework for voice AI user experience.
    Source: github.com/w3c-cg/aikr/blob/main/voiceAI/voice-ai-standards-respec.html
  • Voice AI Standards -- With New Types Editor's Draft
    Editor: Paola Di Maio  •  In progress
    Extended version incorporating new voice AI interaction type classifications. Informed by the W3C Smart Agents Workshop (February 2026).
    Source: github.com/w3c-cg/aikr/blob/main/voiceAI/voice-ai-standards-withnewtypes.html

WebMCP

  • WebMCP Technical Notes Editor's Draft
    Editor: Paola Di Maio  •  In progress
    Technical notes on the W3C Web Machine Learning (WebMCP) specification, covering security considerations, model cards, interoperability requirements, and quality infrastructure for MCP-compliant systems.
    Source: webmcp-technical-notes

Guide to Solid

  • Guide to Solid Editor's Draft
    Editor: W3C AI KR CG  •  In progress
    Solid made simple: a practical introduction to the W3C Solid protocol for content creators, AI agent developers, and users of AI agents. Covers Pod setup, data creation, access control, audit logging, and consent management.
    Source: github.com/w3c-cg/aikr/blob/main/Solid/solidguide.html
